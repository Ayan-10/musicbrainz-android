MusicBrainz is an open music encyclopedia that collects music metadata and makes it available to the public.

Features

This app enables easy access to that data on your Android device. You can scan a barcode to look up release information or search for artists and releases. Registered users can submit tags, ratings and unknown barcodes.

Development

This app is developed by Jamie McDonald. It began as a Google Summer of Code project in 2010. The source code can be found on GitHub. Bug reports and comments are welcomed. Just select 'Feedback' from the menu.

Bugs

You should be aware that error logs are sent from this app in the event of a crash. Personally identifiable information is not included. You can disable this in the settings menu if you like.
